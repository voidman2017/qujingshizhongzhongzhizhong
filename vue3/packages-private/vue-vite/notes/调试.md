# 调试compiler

1. 在项目工程中创建 test 测试文件
2. 测试文件中引入compiler相关模块
3. 执行 `npm run test <测试文件>` 进行测试

或

1. 创建 js 文件
2. 引入 compiler 的dist文件
3. 执行 node debug
